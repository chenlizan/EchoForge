# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/c8f5ddf8-c684-4692-a1f4-62a126c4d3c8

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Ensure local Ollama is running (default: `http://127.0.0.1:11434`) and pull a model (default: `gemma3:12b`)
3. Run the app:
   `npm run dev`
